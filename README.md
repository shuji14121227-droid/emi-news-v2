## Directional atomic layer etching of MgO-doped lithium niobate using Br-based plasma
- **URL**: http://arxiv.org/abs/2511.01825v2
- **AI解析**: 
この半導体論文は、集積フォトニクス分野で重要な非線形光学材料であるニオブ酸リチウム (LiNbO$_3$, LN) の新しいエッチングプロセスについて報告しています。

**要旨のまとめ:**

現在の薄膜ニオブ酸リチウム (TFLN) デバイスは、側壁の表面粗さや形状の不均一性といった作製上の欠陥により性能が制限されており、原子層エッチング (ALE) がその解決策として期待されています。しかし、LNに対する等方性ALEプロセスは存在するものの、パターン転写のための方向性ALEの欠如や、不揮発性化合物の再付着が課題となっていました。

本研究では、LNに対する**方向性原子層エッチング (ALE) プロセス**を開発しました。このプロセスは、表面改質のためのHBr/BCl$_3$/Arプラズマ曝露と、除去のためのArプラズマ曝露を順次行うことで構成されます。

主な成果は以下の通りです。

1.  **再付着の低減:** HBr系ケミストリーは、フッ素系や塩素系プラズマと比較して再付着を低減できることが判明しました。これは、臭素系生成物の蒸気圧が高いことに起因すると考えられます。
2.  **ARDEの抑制:** このプロセスで完全にエッチングされた回折格子パターン（総エッチング深さ220 nm）は、最小の150 nmギャップまでアスペクト比依存エッチング (ARDE) を示しませんでした。これは、同じ深さで300 nmギャップでもARDEが生じるイオンミリングとは対照的です。
3.  **等方性プロセスへの応用:** HBrプラズマケミストリーは、H$_2$プラズマとHBr/BCl$_3$/Arプラズマの順次曝露からなる等方性プロセスもサポートできることが確認されました。

これらの方向性および等方性ALEプロセスを組み合わせることで、TFLNデバイスの作製プロセス全体をALEで完結させることが可能となり、従来のイオンミリングに起因する欠陥を排除できる可能性が示唆されています。

## A transmon qubit realized by exploiting the superconductor-insulator transition
- **URL**: http://arxiv.org/abs/2510.19983v1
- **AI解析**: 
この論文は、**半導体ではなく、超伝導量子ビット**に関するものです。以下にその要約を日本語で示します。

---

**論文タイトル:** 超伝導体-絶縁体転移を活用して実現されたトランスモン量子ビット

**要約:**

超伝導量子ビットは、実用的な量子コンピュータを実現するための最も有望なプラットフォームの一つです。量子プロセッサを構築するために必要な非線形性は、通常、2つのアルミニウム電極の間に酸化アルミニウム層を挟むことで形成されるジョセフソン接合によって達成されます。しかし、これらの接合にはいくつかの限界があり、スケーラビリティを妨げています。具体的には、アルミニウムの超伝導ギャップが小さいためミリケルビンでの動作が必要であること、材料界面が散逸を引き起こすこと、そしてサンドイッチ構造が高周波アプリケーションにとって不必要な容量を生み出すことです。

本研究では、これら3つの限界すべてに対処するため、**超伝導体-絶縁体転移（SIT）を活用した新しい超伝導弱結合（weak link）**を提案しています。ニオブナイトライド（NbN）の単一薄膜を局所的に薄くすることで、その厚さに依存する超伝導体-絶縁体転移を利用し、原子層堆積（ALD）と原子層エッチング（ALE）のみを用いて弱結合を形成しました。

この新しい弱結合を用いて、「プラナロン」と名付けられたトランスモン量子ビットを生成し、**非調和性 $α/2π= 235$ MHz** を測定しました。現在の線幅は $κ/2π= 15$ MHz です。

ニオブナイトライドの高い超伝導ギャップは、将来のデバイスにおいてより高温での動作を可能にする可能性があり、また、弱結合の完全に平面的な形状は、不要な材料界面や容量を排除します。SIT近傍の小さな材料領域の調査は、散逸や有限サイズ効果を含む転移の性質に関する新たな知見をもたらす可能性も秘めています。

## Atomic Layer Etching of Aluminum Nitride: Mechanistic Insights from First-Principles Studies of Chlorine Chemistry
- **URL**: http://arxiv.org/abs/2509.07554v1
- **AI解析**: 
この半導体論文は、窒化アルミニウム（AlN）の原子層エッチング（ALE）プロセスを最適化するために、塩素化学（原子状塩素および塩素分子）がAlN表面とどのように相互作用するかを、第一原理計算を用いて調査しています。

**主要な内容と発見:**

*   **研究対象と手法:** AlNのAl終端0001表面に着目し、第一原理密度汎関数理論（DFT）計算とクライミングイメージ・ナッジドエラスティックバンド（CI-NEB）法を組み合わせて、原子状塩素（Cl）と塩素分子（Cl2）の吸着、脱着、拡散挙動を調べました。
*   **化学吸着の特性:** 研究の結果、原子状ClとCl2の両方がAlN表面に対して高い結合エネルギーを持つ化学吸着を示すことが明らかになりました。
*   **塩素分子の分解:** 特に重要な発見として、Al終端AlN 0001表面におけるCl2分子の分解経路が、エネルギー障壁のない（バリアフリーな）発熱プロセスであることが示されました。これは、Cl2がAlN表面に容易に反応し、分解してエッチング反応に寄与することを示唆しています。

**結論と意義:**

これらの微視的スケールでの知見は、ハロゲンと絶縁体表面の相互作用に関する新たな理解を提供します。これにより、半導体製造におけるAlNの原子層エッチングプロセスを最適化するための新たな戦略を立てる機会が生まれると期待されます。
