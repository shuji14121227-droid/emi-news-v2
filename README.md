## Directional atomic layer etching of MgO-doped lithium niobate using Br-based plasma
- **URL**: http://arxiv.org/abs/2511.01825v2
- **AI解析**: 
この半導体論文は、集積フォトニクス分野で重要な非線形光学材料であるニオブ酸リチウム (LiNbO$_3$, LN) の加工における課題と、それを克服するための新たな原子層エッチング (ALE) プロセスについて報告しています。

**要約:**

ニオブ酸リチウム (LN) は、光学通信から量子情報処理まで幅広い応用を持つ集積フォトニクス向けの材料です。しかし、薄膜ニオブ酸リチウム (TFLN) デバイスの性能は、側壁表面の粗さや形状の不均一性といった製造上の欠陥によって現在制限されています。これらの課題を解決するために原子層エッチング (ALE) が期待されていますが、LNに対する等方性ALEプロセスは報告されているものの、パターン転写のための異方性ALEプロセスの欠如や、不揮発性化合物による再付着が課題となっていました。

本研究では、LNに対する**異方性ALEプロセス**を開発しました。このプロセスは、HBr/BCl$_3$/Arプラズマによる表面改質と、それに続くArプラズマによる除去という連続的な工程から構成されます。

主要な発見は以下の通りです。

1.  **再付着の低減:** HBrベースの化学反応を用いることで、フッ素系や塩素系プラズマと比較して再付着が低減されることが示されました。これは、臭素系生成物の蒸気圧が高いことに起因すると考えられます。
2.  **優れたエッチング特性:** このプロセスを用いて完全にエッチングされた回折格子パターン（合計エッチング深さ220 nm）は、最小150 nmのギャップに至るまで**アスペクト比依存性エッチング (ARDE) を示しませんでした**。これは、同じエッチング深さで300 nmのギャップでもARDEが現れるイオンミリングとは対照的です。
3.  **等方性プロセスのサポート:** HBrプラズマの化学反応は、H$_2$プラズマとHBr/BCl$_3$/Arプラズマの連続的な曝露からなる等方性プロセスもサポートすることが分かりました。

これらの異方性および等方性ALEプロセスは、TFLNデバイスの製造プロセスに統合され、イオンミリングに起因する加工欠陥を排除し、デバイス性能を向上させる可能性を秘めています。

## A transmon qubit realized by exploiting the superconductor-insulator transition
- **URL**: http://arxiv.org/abs/2510.19983v1
- **AI解析**: 
この半導体論文は、「超伝導-絶縁体転移（SIT）を活用して実現されたトランスモン型量子ビット」について述べています。

以下に要約します。

**タイトル:** 超伝導-絶縁体転移を活用したトランスモン型量子ビットの実現

**要約:**

超伝導量子ビットは、実用的な量子コンピュータを実現するための最も有望なプラットフォームの一つです。量子プロセッサを構築するには非線形性が必要であり、超伝導回路では通常、2つのアルミニウム電極間に酸化アルミニウム層を挟むことでジョセフソン接合を形成し、これによって非線形性を得ています。

しかし、これらの従来のジョセフソン接合にはスケーラビリティを阻害するいくつかの課題があります。
1.  アルミニウムの超伝導ギャップが小さいため、ミリケルビンという極低温での動作が必要になります。
2.  材料界面が存在することでエネルギー散逸が生じます。
3.  サンドイッチ構造は、高周波用途において不要な容量を増加させます。

本研究では、これらの3つの課題全てに対し、超伝導-絶縁体転移（SIT）を利用した新しい超伝導弱結合を用いることで対処しています。具体的には、窒化ニオブ（NbN）の単一薄膜を局所的に薄膜化し、その膜厚に依存するSITを活用することで弱結合を形成しました。このプロセスには、原子層堆積（ALD）と原子層エッチング（ALE）のみが用いられています。

この弱結合を用いて、「planaron」と名付けられたトランスモン型量子ビットを作製しました。測定された非調和性は $\alpha/2\pi = 235$ MHz であり、現在の線幅は $\kappa/2\pi = 15$ MHz です。

NbNの高い超伝導ギャップは、将来的にデバイスの高温動作を可能にする可能性があり、弱結合の完全に平面的な構造は、不要な材料界面と容量を排除します。また、SIT近傍の微小な材料領域を調査することで、散逸や有限サイズ効果など、この転移の性質について新たな知見が得られる可能性も示唆されています。

## Atomic Layer Etching of Aluminum Nitride: Mechanistic Insights from First-Principles Studies of Chlorine Chemistry
- **URL**: http://arxiv.org/abs/2509.07554v1
- **AI解析**: 
この半導体論文は、第一原理密度汎関数理論（DFT）計算とクライミングイメージヌージドエラスティックバンド法（CI-NEB）を組み合わせて、窒化アルミニウム（AlN）のAl終端0001表面における原子塩素（Cl）および分子塩素（Cl2）の吸着、脱着、拡散について調査しました。

主な結果として、原子Clと分子Cl2の両方が高い結合エネルギーを持つ化学吸着を示すことが明らかになりました。さらに、AlNのAl終端0001表面でのCl2の分裂経路が、活性化障壁がなく（barrierless）、発熱的（exothermic）なプロセスであることも判明しました。

これらの知見は、ハロゲンと絶縁体の表面相互作用に関する新しい微視的な洞察を提供し、半導体製造におけるAlNの原子層エッチング（ALE）プロセスを最適化するための新たな戦略に貢献する可能性を秘めています。
